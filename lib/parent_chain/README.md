# Parent Chain Module

This module provides the foundation for CoinShift 1.0's multi-chain swap functionality.

## Structure

- **`config.rs`**: Configuration for parent chain node connections
- **`client.rs`**: Abstract client interface for parent chain RPC interactions
- **`swap.rs`**: Trustless swap implementation with conditional payments

## Usage

See `COINSHIFT.md` in the project root for detailed usage and architecture documentation.
